About Meme Bot

This bot is designed to provide users with services that make their everyday tasks more convenient.

According to the requirements of the final project, the Meme Bot is written in pure Flask without using specialized libraries for Telegram. It is built on the interaction with the TelegramBot API. This project consists of only one endpoint through which all requests are processed. For data storage, SQLite database is used. In case of project scaling, it is recommended to use PostgreSQL. During the development process, principles of OOP, DRY, SOLID, and RESTful were applied.
Here is detailed information about the available services:

  • Memes - It is scientifically proven that laughter prolongs life. Therefore, the main function of this bot is to provide the user with an almost unlimited number of memes. It utilizes a third-party Programming Memes Images API that generates random IT-themed memes. Approximately 1800 memes are available every day. To access a larger quantity, a paid subscription needs to be obtained.
  • Weather - It provides the user with the current weather in the specified city. The Geocoding API is used to determine the coordinates of the city. The obtained coordinates are then passed to the Weather Forecast API. The user receives data such as wind speed, wind direction, and air temperature. The number of indicators can be significantly expanded as needed.
  • Phonebook - Everyone knows what a phone book is. This, along with the Weather service, are mandatory requirements for the project. The following capabilities have been implemented: saving a new contact, retrieving contact information, and deleting a specific contact. The service has been implemented in such a way that each user has access only to their own contacts.
  • Commands - Another mandatory requirement for the project was to provide the user with a list of available commands for the bot. The available commands are set using the setWebhook method of the TelegramBot API.

Available commands FAQ

Here is a list of commands available to users of this bot's Menu button.

  • /memes

    This command sends the message to the user with a randomly generated IT-themed meme.

  • /weather

    This command displays the current weather data for the specified city. First, the bot will ask for the city name. After that, the user selects the desired option from the provided search results list.

  • /commands

    When selecting this command, the user receives a message with a list of all available commands.

  • /add_contact

    The functionality of this command allows the user to add a new contact to their phone book.

  • /get_contact

    This command provides the user with information about the specified contact. If there are multiple matches in the phone book for the user's query, the screen will display information about all those contacts.

  • /delete_contact

    When selecting this command, the user can delete a contact from their phone book. It's important to note that the user's request must exactly match the name of the contact for the deletion to occur.
